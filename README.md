# Tapbotic AI Pvt Ltd - Corporate Website

This is the official corporate website for Tapbotic AI Pvt Ltd, an enterprise-grade Artificial Intelligence, Automation, and Software Services company. The website is designed to showcase the company's services, leadership, expertise, and provide a point of contact for potential clients, partners, and job applicants.

---

## Features

- **Responsive Design:** Designed to provide a great experience on all devices (desktop, tablet, and mobile).
- **Service Pages:** Detailed information on services like AI, Automation, Software Development, Cloud & DevOps, Cyber Security, etc.
- **Company Information:** Important information about the company such as 'About Us', 'Governance & Leadership', 'Careers'.
- **Blog Section:** Articles on company expertise and latest industry trends.
- **Interactive UI:** Animations, sliders, and interactive elements for an engaging user experience.
- **Contact Form:** A PHP-based contact form for users to easily get in touch.
- **Dark Theme:** Dark mode to provide a comfortable viewing experience.

---

## Setup and Installation

To run this website on your local machine, follow the steps below:

1.  **Clone the repository:**
    ```sh
    git clone <repository-url>
    ```

2.  **Local Server:**
    - You will need a local server environment like XAMPP, WAMP, or MAMP.
    - Place the project files in the server's root directory (e.g., `htdocs` for XAMPP).

3.  **Start Server:**
    - Start the Apache server from your XAMPP/WAMP/MAMP control panel.

4.  **Access the Website:**
    - Open your web browser and go to the following URL:
      ```
      http://localhost/Tapbotic_AI_Pvt_Ltd/html/
      ```
